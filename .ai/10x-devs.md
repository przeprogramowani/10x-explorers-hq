# 10xDevs 3.0 — Specification

## Overview

**10xDevs** is a 5+1 week intensive program by Przeprogramowani & BRAVE (publishers of AI_devs) that teaches developers how to work with AI across the entire software development lifecycle — from research and planning, through implementation, to production maintenance.

- **Edition:** 3.0
- **Start date:** 18.05 (2025/2026)
- **Duration:** 5+1 weeks (5 core modules + bonus week)
- **Alumni:** 3,700+ (across all editions)
- **Rating:** 5 stars

## Core Philosophy

The program is built around a **repeatable, production-grade process** for working with AI agents — not prompt tricks or demos. Key principles:

- AI works as an engineering tool, not a chatbot
- Focus on real repos, real APIs, complex functions, real problems
- No guessing, no hallucinations, no chaos
- Context Engineering as a foundation (context windows, drift, summaries, thread management, Agents.md files)
- Full AI-Native SDLC

## Program Structure

Learning tracks within each module:

- **Core** — quick entry, essentials only
- **Deep Dive** — full technical breakdown
- **Space Explorers** — interactive quests in 2D Game

---

## Module 1 — Agentic Environment

**Goal:** Set up a fully configured AI-powered development environment.

### Lessons

1. Od pomysłu do PRD: Metoda Sokratejska z Agentem
2. Od chatbota do Agenta: toolkit, bootstrap i delegacja
3. Agent pod kontrolą: uprawnienia, izolacja, bezpieczeństwo
4. Nowa wiedza Agenta: MCP, Skills i kontekst projektowy
5. Innovate: Evale na Twoim stacku - stwórz benchmark dla modeli AI

---

## Module 2 — 10xDevs Workflow

**Goal:** Deliver a working MVP feature-by-feature using a repeatable AI workflow.

### Lessons

1. Plan MVP: milestony, zależności i bezwzględne priorytety
2. Architektura z Agentami: zaplanuj pracę, zanim wygenerujesz kod
3. Implementacja z AI: 80% ready challenge i kontrola kontekstu
4. Solo Code Review: weryfikuj kod AI szybko i skutecznie
5. Innovate: Więcej ficzerów, mniej czekania - wielowątkowa praca z Agentami

---

## Module 3 — AI Development Quality & Maintenance

**Goal:** Build a testing and debugging infrastructure powered by AI agents.

### Lessons

1. Plan testów z AI: quality gates, testing guide i priorytety
2. Od planu do testów: implementacja unitów z Agentem
3. Hooki i triggery: Agent, który sam reaguje na błędy
4. Testy E2E: Playwright, MCP i multimodalne scenariusze
5. Debugowanie z AI: od stack trace'a do gotowego fixa

---

## Module 4 — Large Scale & Legacy Projects

**Goal:** Scale AI context for complex legacy systems and drive modernization.

### Lessons

1. Nowy-stary projekt? Agent zbuduje Ci mapę i wyjaśni architekturę
2. Analiza feature z AI: co działa, co kuleje, co zmodernizować
3. Refaktoryzacja z Agentem: testy, zmiany, weryfikacja
4. Modernizacja legacy z DDD: wydzielaj domeny, potem deleguj Agentowi
5. Skalowanie kontekstu dla AI w dużych projektach

---

## Module 5 — AI-Native Teamwork

**Goal:** Integrate AI into team engineering processes and build internal AI tools.

### Lessons

1. Twój pierwszy Agent zespołowy: SDK, koszty, metryki
2. Code Review w erze AI: standardy, DoD i Agent w pipeline
3. Shared AI Registry: skille, komendy i reguły dla zespołu
4. AI Internal Builders: wewnętrzne narzędzia, serwisy i automatyzacje
5. Innovate: Async & Remote Agents - deleguj i zajmij się czymś innym
